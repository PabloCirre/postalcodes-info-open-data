# PostalCodes.info Data and API Notes

PostalCodes.info publishes postal-code reference pages, dataset documentation, and API integration notes for country, region, and locality workflows.

Useful links:

- Dataset documentation: https://postalcodes.info/datasets
- API documentation: https://postalcodes.info/api
- Canonical site: https://postalcodes.info/
- Zenodo dataset DOI: https://zenodo.org/records/19493709
- Zenodo methodology note: https://zenodo.org/records/19930578
- Zenodo coverage benchmark: https://zenodo.org/records/19930688

## Use cases

The reference material is intended for address-form testing, postal-code lookup examples, checkout validation, logistics normalization, data quality checks, and geospatial workflows.

## Current public references

The dataset page documents country-level postal-code downloads and field guidance. The API page documents lookup patterns, JSON access, search examples, and implementation notes.

## Citation

Use the Zenodo DOI records above when a citable public reference is needed.
