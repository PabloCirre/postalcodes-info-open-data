# PostalCodes.info Open Data Samples

PostalCodes.info maintains postal-code reference pages, dataset notes, and API documentation for country, region, and locality postal-code workflows.

Useful links:

- Dataset documentation: https://postalcodes.info/datasets
- API documentation: https://postalcodes.info/api
- APIs.json metadata: https://raw.githubusercontent.com/PabloCirre/postalcodes-info-open-data/main/apis.json
- Canonical site: https://postalcodes.info/
- Zenodo methodology note: https://zenodo.org/records/19930578
- Zenodo coverage benchmark: https://zenodo.org/records/19930688

## What this repository is for

This repository is a public pointer for reusable PostalCodes.info dataset samples, validation notes, and examples. It is intended for developers and data teams who need a neutral reference for address-form testing, postal-code lookup examples, data quality checks, and geospatial workflows.

## Current data references

The primary public dataset documentation lives at https://postalcodes.info/datasets. The API reference lives at https://postalcodes.info/api.

The benchmark note documents how PostalCodes.info checks URL health, canonical consistency, sitemap eligibility, demand, internal linking support, and monetization readiness across postal-code pages.
